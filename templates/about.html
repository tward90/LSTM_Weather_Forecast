Project Objective

To apply machine learning models to the NOAA_GSOD dataset to forecast 5 days of Houston weather given current weather conditions. Furthermore, to create an application that will dynamically display the forecast in a web page.

Data Overview

NOAA GSOD - Global Surface Summary of the Day Weather

DataSource : Kaggle: NOAA_GSOD

This public dataset was created by the National Oceanic and Atmospheric Administration (NOAA) and includes global data obtained from the USAF Climatology Center. This dataset covers GSOD data between 1929 and present (updated daily), collected from over 9000 stations. Global summary of the day is comprised of a dozen daily averages computed from global hourly station data. Daily weather elements include mean values of: temperature, dew point temperature, sea level pressure, station pressure, visibility, and wind speed plus maximum and minimum temperature, maximum sustained wind speed and maximum gust, precipitation amount, snow depth, and weather indicators. Except for U.S. stations, 24-hour periods are based upon UTC times.

Data ETL

The data was extracted using BigQuery. The first step being that of identifying weather stations in the Houston area, then querying the datasets for each year for each station.
